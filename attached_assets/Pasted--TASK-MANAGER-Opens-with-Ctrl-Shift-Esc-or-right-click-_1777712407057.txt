

**TASK MANAGER**

Opens with Ctrl+Shift+Esc or right-click taskbar. Tabs: Processes (table with Name, Status, CPU%, Memory columns — CPU and Memory values animate realistically, sorted by CPU by default, click column headers to sort), Performance (sidebar: CPU, Memory, Disk, GPU — click each to show animated graph using Canvas — CPU shows sine-wave-ish animated usage graph, Memory shows used/available bar and composition), App History, Startup (list of startup apps with Impact rating, Enable/Disable right-click toggle), Users, Details, Services (static list of fake services). End Task button for selected process closes the corresponding app window. Right-click a process: End Task, End Process Tree, Set Priority (simulated), Open File Location. Search/filter bar filters process list. Always on top toggle.

**SEARCH OVERLAY**

Windows+S or clicking taskbar search bar opens full search overlay. Blurred dark background. Search input auto-focused. As user types: live results appear below in categorized sections — Apps (matches installed app names, click to open), Files (matches filenames in virtual FS, click to open), Settings (matches settings panel names, click to open), Web (shows "Search the web for [query]" that opens Error64 Browser with a Google search URL). Recent searches shown when search box is empty. Keyboard navigation with Up/Down arrow and Enter to open result. Escape closes overlay.

**TASK VIEW AND VIRTUAL DESKTOPS**

Windows+Tab or Task View button opens the Task View overlay. All open windows shown as large draggable thumbnails with live content previews. Click thumbnail to focus that window. Close button on each thumbnail. Virtual Desktops bar at the top: current desktops shown as numbered cards, + New Desktop button. Click a desktop to switch (open windows isolated per desktop). Drag a window thumbnail to a different desktop card to move it. Right-click desktop card: Rename, Remove. Switching desktops animates with a horizontal slide.

**KEYBOARD SHORTCUTS — ALL MUST WORK**

Windows key: toggle Start Menu. Windows+D: show desktop (minimize all) / restore all. Windows+E: open File Explorer. Windows+L: lock screen. Windows+R: open Run dialog. Windows+S: open Search. Windows+A: open Notification Center. Windows+Tab: Task View. Windows+Left/Right/Up/Down: snap and maximize active window. Windows+Shift+S: Snipping Tool capture mode. Windows+Plus/Minus: Magnifier zoom. Windows+Escape: close Magnifier. Alt+Tab: app switcher overlay. Alt+F4: close active window. Ctrl+Shift+Esc: Task Manager. F1: open Help (static About page). F2: rename selected item. F5: refresh. F11: fullscreen active window. F12: toggle browser DevTools note. Delete: delete selected. Escape: close overlays, deselect, cancel. Ctrl+Z: undo. Ctrl+Y: redo. Ctrl+C: copy. Ctrl+X: cut. Ctrl+V: paste. Ctrl+A: select all. Ctrl+N: new (in apps). Ctrl+S: save (in apps). Ctrl+O: open (in apps). Ctrl+F: find (in apps). Ctrl+P: print (simulated). Ctrl+T: new tab (in browser). Ctrl+W: close tab (in browser). Ctrl+R: refresh (in browser).
