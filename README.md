# Error64

A browser-based Windows 10 simulation. Runs entirely in the browser with no backend.


## Tech Stack

- React 19
- Vite 7
- TailwindCSS 4
- Radix UI (components)
- Framer Motion (animations)
- wouter (routing)
- zod (validation)

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Virtual desktop with icons, taskbar, start menu
- Window management (drag, resize, minimize, maximize)
- Multiple built-in apps (Browser, Settings, Mail, Photos, Media Player, Calendar, Notepad, Command Prompt, Task Manager, etc.)
- Virtual file system (persisted to IndexedDB)
- System settings (themes, wallpapers, etc.)
